### Getting Started

macOS

```
## Install Node
// if homebrew is already installed
brew update
brew upgrade node
node -v

// else
brew update
brew install node
node -v

## Clone repo locally
git clone https://github.com/chani-nicholas-inc/api-technical-interview.git
npm i
npm i -g nodemon
nodemon
```
